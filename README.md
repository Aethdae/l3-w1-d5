# Read/Write terminal file

A small terminal app to add content to a file, and can read files given their path.

## Usage:

```
npm start [commands...]
```

Runs app with the given commands.

## Commands:

```
append (alias: a) "string"
```

Appends input string to the output file.

```
read (alias: r) "path" (Default: output.txt)
```

Reads the file at the given path. Defaults to the file created by the app.

## Installation:

```
npm i
```

Installs required packages.
